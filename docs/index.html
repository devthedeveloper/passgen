Usage

Quick mode
# Segmented password with dash separator (5Ã—5 chars)
passgen -
nJ0aK-6V96F-zMtQs-o8kuh-sZty9

# Same but with underscore separator
passgen _
mrwlN_XfZNo_awT1y_wDrPk_JY6X9
Random passwords
passgen -length 32
passgen -count 5
passgen -no-symbols
passgen -exclude "0OIl1"          # remove ambiguous chars
Segmented passwords
passgen -type segment
passgen -type segment -segments 4 -seg-length 6
passgen -type segment -separator _
passgen -type segment -no-upper   # lowercase + digits only
Interactive mode
passgen
# Walks you through all options step by step
